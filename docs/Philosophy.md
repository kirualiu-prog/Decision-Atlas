# Philosophy

Decision Atlas is built on one practical belief:

> The value of a thinking tool is not whether it sounds intelligent, but whether it improves judgment under real constraints.

## What This Project Is

Decision Atlas is a structured knowledge base for thinking tools, decision science, mental models, systems thinking, strategy, behavioral economics, and practical reasoning.

It is designed to help users answer three questions:

1. What is the nature of the problem?
2. Which tool should be used?
3. In what sequence should tools be applied?

## What This Project Is Not

Decision Atlas is not:

- a list of clever concepts;
- a replacement for domain expertise;
- a collection of quotes;
- a copy of any private course material;
- a shortcut for avoiding judgment.

## Writing Principles

Each entry should be:

- Original: rewritten in its own structure and language.
- Precise: focused on the core logic of the tool.
- Practical: useful for actual decisions.
- Connected: linked to related and prerequisite tools.
- Limited: clear about what the tool cannot solve.

## Core Standard

A model is only included if it helps at least one of the following:

- reduce cognitive error;
- structure uncertainty;
- clarify trade-offs;
- reveal causes;
- map systems;
- improve strategic judgment;
- guide action under constraints.
