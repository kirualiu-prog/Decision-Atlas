# Decision Guide

This guide maps common thinking problems to recommended tools.

The goal is to help users start from the problem, not from the model.

## I do not know whether the information is reliable

Use:

- DS-001 Base Rate
- DS-002 Bayesian Updating
- DS-007 Regression to the Mean
- DS-015 Confidence Interval
- DS-016 Confirmation Bias
- DS-027 Survivorship Bias
- DS-097 Signal vs Noise

Core question:

> What evidence should update my belief, and by how much?

## I have too many options and cannot choose

Use:

- DS-041 Opportunity Cost
- DS-042 Trade-off
- DS-043 Decision Matrix
- DS-044 Decision Tree
- DS-047 Cost-Benefit Analysis
- DS-049 Marginal Thinking
- DS-050 Reversible vs Irreversible Decisions

Core question:

> What am I giving up, and which choice creates the best expected outcome under current constraints?

## I do not know the real cause of the problem

Use:

- DS-031 First Principles
- DS-032 Five Whys
- DS-033 Root Cause Analysis
- DS-034 MECE
- DS-035 Issue Tree
- DS-036 Constraint Theory
- DS-037 Bottleneck Analysis

Core question:

> What must be true for this problem to exist?

## I worry about unintended consequences

Use:

- DS-048 Second-order Thinking
- DS-053 Premortem
- DS-055 Scenario Planning
- DS-056 Systems Mapping
- DS-057 Feedback Loop
- DS-061 Delay
- DS-063 Leverage Point

Core question:

> If this works, what happens next? If this fails, why?

## I need to understand a complex system

Use:

- DS-056 Systems Mapping
- DS-057 Feedback Loop
- DS-058 Reinforcing Loop
- DS-059 Balancing Loop
- DS-060 Stocks and Flows
- DS-061 Delay
- DS-062 Emergence
- DS-067 Complex Adaptive Systems

Core question:

> What relationships, feedback loops, and delays are shaping the outcome?

## I need to make a strategic choice

Use:

- DS-071 Competitive Advantage
- DS-072 Flywheel
- DS-075 Value Chain
- DS-076 Porter's Five Forces
- DS-077 Game Theory
- DS-080 Optionality
- DS-081 Incentives

Core question:

> Where is advantage created, protected, or lost?

## I think people are behaving irrationally

- DS-081 Incentives
- DS-084 Social Proof
- DS-085 Mental Accounting

- DS-088 Identity Economics

## I need better ideas

Use:

- DS-091 Divergent Thinking

- DS-093 Lateral Thinking
- DS-094 Design Thinking
- DS-095 Jobs to Be Done

## I need to challenge my own thinking

- DS-098 Inversion
