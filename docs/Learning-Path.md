# Learning Path

Decision Atlas should not be read only from DS-001 to DS-100.

The recommended path starts with tools that improve basic judgment, then moves into analysis, decision-making, systems, strategy, and meta-thinking.

## Usage Rule

When facing a real problem, do not ask, "Which model do I like?"

Ask:

1. Is this an information problem?
2. Is this a judgment problem?
3. Is this a root-cause problem?
4. Is this a trade-off problem?
5. Is this a system problem?
6. Is this a strategy problem?
7. Is this a people-behavior problem?

Then choose tools accordingly.
