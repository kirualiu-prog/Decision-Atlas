# Decision Atlas

Decision Atlas is a Markdown-first knowledge base for decision science, thinking tools, and practical judgment.

Current status:

- The full `DS-001` to `DS-100` model set now exists as structured first-pass drafts.
- The taxonomy, learning path, decision guide, and dependency map are in place.
- The structural editorial pass is complete across the full library.
- The next phase is content polish: tighter examples, smoother tone, and more even depth.

## Quick Routes

- If you want the fastest way to find a tool for a live problem, start with [`docs/Decision-Guide.md`](./docs/Decision-Guide.md).
- If you want a recommended reading order, start with [`docs/Learning-Path.md`](./docs/Learning-Path.md).
- If you want to browse the full library by category, start with [`docs/Model-Index.md`](./docs/Model-Index.md).
- If you want to turn a raw question into a model-driven answer, use [`docs/Question-Analysis-Workflow.md`](./docs/Question-Analysis-Workflow.md) and [`templates/Question-Answer-Prompt.md`](./templates/Question-Answer-Prompt.md).
