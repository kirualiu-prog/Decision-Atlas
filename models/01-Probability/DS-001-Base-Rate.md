# DS-001 Base Rate（基準率）

> 真正好的判斷，永遠從母體開始，而不是故事。

## 一句話定義

Base Rate（基準率）是某件事情在整體母體中原本發生的比例，也是任何機率判斷最應該先考慮的資訊。

## 核心概念

人在做判斷時，很容易被眼前看到的案例、新聞、故事或個人經驗吸引。

但是：

> 個案不是世界。

真正代表世界的是母體。Base Rate 就是在回答：

> 如果完全不知道眼前這個特殊案例，只看整體世界，它原本發生的機率是多少？

## 核心邏輯

所有判斷都有兩種資訊：

第一種是母體本來長什麼樣子。

第二種是新出現的證據。

Base Rate 就是第一種。如果沒有 Base Rate，任何新證據都容易被過度解讀。

## 為什麼重要

基準率能幫你在故事、情緒和個案之前，先把判斷放回母體。

很多錯誤不是因為不會思考，而是因為太快被眼前案例帶走。

## 使用時機

適合用在這些情境：

- 你正在根據少數案例判斷整體趨勢
- 你需要估計某件事原本有多常發生
- 你懷疑自己被新聞、故事或近期事件影響

## Key Questions

- 這個模型在這個情境真的適用嗎？
- 它最重要的前提或限制是什麼？
- 如果用這個模型看問題，決策會因此改變什麼？

## Failure Modes

- 把模型套到不適合的問題
- 把模型當成唯一答案
- 記住名詞，卻沒有回到實際判斷

## Concept Boundary

這個模型提供的是一種特定視角，不會單獨取代完整分析。

比較好的用法，通常是和相鄰模型一起交叉檢查。
