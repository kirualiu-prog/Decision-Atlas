# DS-066 Network Effect（網路效應）

> 有些價值不是產品本身帶來的，而是有多少人一起在裡面。

## 一句話定義

Network Effect（網路效應）是使用者越多，產品或系統對每個使用者的價值也越高的現象。

## 核心概念

社交平台、市集、支付網路、協作工具，都可能靠網路效應成長。

一旦形成，早期差距會被放大；但若沒跨過門檻，也可能一直起不來。

## 核心邏輯

每新增一個節點，就讓其他節點的互動與效用增加。這形成增強迴路，常會帶來贏者更贏的局面。

## 為什麼重要

網路效應能解釋很多平台型成長為何不是線性的。

它也提醒你，某些系統的價值不是內建在產品裡，而是長在關係裡。

## 使用時機

適合用在這些情境：

- 你在分析平台、社群或雙邊市場
- 某個產品價值會隨參與者增加而變高
- 你需要判斷成長是否有自我加速結構

## Key Questions

- 這個模型在這個情境真的適用嗎？
- 它最重要的前提或限制是什麼？
- 如果用這個模型看問題，決策會因此改變什麼？

## Failure Modes

- 把模型套到不適合的問題
- 把模型當成唯一答案
- 記住名詞，卻沒有回到實際判斷

## Concept Boundary

比較好的用法，通常是和相鄰模型一起交叉檢查。
